Projet d'application android pour le château de fougères

XAML • C# • Markdown

Nombre de personnes sur le projet : 1

Présentation du projet

Le projet a été réalisé par moi-même, le code est accessible sur GitHub : dépot GitHub

Origine et besoin client

Le projet provient d’une demande du château de fougères. Il souhaitait avoir une application quelconque qui permet sans connexion internet de consulter les vidéos de présentation en langue des signes française pour les personnes malentendantes, avant ils avaient un site web local a déployer sur tablette puis ils devait aller sur internet pour consulter les vidéos, sauf que la connexion internet n'est pas stable, ils m'ont donc demander de trouver une solution, j'ai donc proposer de faire une application android.

Phase de conception

Un brainstorming a été réalisé afin de définir les fonctionnalités et le design. Cette phase s’est déroulée sur une journée, la personne n'a pas demander d'interface utilisateur particulière, elle souhaitais juste qu'on ai accès aux vidéos hors ligne et qu'on puisse cliquer sur la carte.
